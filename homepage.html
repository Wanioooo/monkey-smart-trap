System Overview

The Monkey SmartTrap system is a web-based intelligent prototype designed to support wildlife monitoring. It utilises computer vision techniques to detect the presence of monkeys in monitored trap areas using camera feeds within a controlled or simulated environment.

Intelligent Detection

Computer vision algorithms analyse video frames to identify monkeys. Detection is limited exclusively to monkeys as a proof of concept.

Real-Time Insights

The system provides near real-time detection feedback to assist monitoring without physical deployment or live trap testing.

Secure Access

Access to the system is restricted to authorised users only, ensuring controlled and secure monitoring operations.

Controlled Testing

All testing is conducted in simulated environments. The system serves as a research-focused prototype for wildlife monitoring.
